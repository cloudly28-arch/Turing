Графический эмулятор машины Тьюринга с визуализацией ленты, состояний и пошаговым выполнением алгоритмов.

## 📋 Особенности

- **Визуальный редактор ленты** — создание и редактирование символов на ленте
- **Редактор команд** — задание правил переходов (состояние, символ → новый символ, направление, новое состояние)
- **Пошаговое выполнение** — контроль выполнения алгоритма шаг за шагом
- **Автоматический режим** — непрерывное выполнение с регулируемой скоростью
- **Визуализация** — подсветка текущего состояния и позиции головки
- **Сохранение/загрузка** — сохранение программ машины Тьюринга в файлы (JSON)
- **Примеры алгоритмов** — встроенные примеры (унарное сложение, копирование строки и др.)
